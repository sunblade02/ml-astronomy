# ML Astronomy

ML Astronomy is a machine learning project for stellar classification.  
It predictsthree types of astronomical objects from spectral characteristics:
- Star
- Quasar (Quasi stellar / Supermassive black hole)
- Galaxy

## Datasets

- **Stellar Classification Dataset - SDSS17** from kaggle : https://www.kaggle.com/datasets/fedesoriano/stellar-classification-dataset-sdss17
